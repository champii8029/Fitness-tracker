# 💪 CHAMPII — Annual Fitness & Keto Planner

Planificador anual personalizado de **Calistenia**, **Pilates** y **Dieta Keto** con seguimiento de progreso, IMC y calendario interactivo.

## ✨ Características

- 📅 **365 días** de plan progresivo (6 fases de dificultad)
- 💪 **Calistenia** con peso corporal — de principiante a élite
- 🧘 **Pilates** enfocado en core y estabilidad lumbar (L5-S1)
- 🥑 **Dieta Keto** — recetas + alimentos permitidos + macros diarios (70F/25P/5C)
- 📐 **Calculadora de IMC** en tiempo real
- 🏆 **Gamificación** — felicitaciones y guía al siguiente ejercicio
- 📊 **Gráficas de progreso** — tendencia de peso y ejercicios
- 🔗 **Código de sincronización** — comparte datos entre dispositivos
- 📱 **100% responsive** — funciona perfecto en celular

## 🚀 Uso

Abre `index.html` directamente en tu navegador. ¡No requiere servidor ni instalación!

### Sincronizar entre dispositivos
1. Haz clic en **🔗 Sync** en la parte superior
2. El código se copia automáticamente al portapapeles
3. En el otro dispositivo, haz clic en **🔗 Sync** y pega el código

### Backup
- **⬇ Backup** → Descarga un archivo JSON con todos tus datos
- **⬆ Importar** → Carga un backup previo
